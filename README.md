# أثَر — Achievement OS

حوّل العمل اليومي المبعثر إلى منجزات مهنية موثقة، قابلة للبحث، وجاهزة للسيرة الذاتية أو الترقية أو البورتفوليو أو المقابلات.

## النسخة الحالية

هذه نسخة MVP محلية بالكامل؛ لا تحتاج حسابًا أو API مدفوعًا، ولا ترسل المحتوى إلى أي خادم.

- التقاط الإنجاز عبر رحلة إرشادية من 3 مراحل.
- تقييم الجاهزية من 100 بدون اختراع نتائج أو أرقام.
- كشف المعلومات والأدلة الناقصة.
- مكتبة منجزات مع البحث والفلاتر.
- تعديل وحذف الإنجازات وحفظها في `localStorage`.
- توليد عربي وإنجليزي لخمس حالات: CV، تنفيذي، ترقية، بورتفوليو، ومقابلة STAR.
- خريطة مهارات مبنية على الأدلة المسجلة.
- تصميم RTL متجاوب وقابل للتثبيت كتطبيق ويب.

## التشغيل

```bash
npm install
npm run dev
```

## التحقق

```bash
npm run test
npm run build
```

## الخصوصية

البيانات محفوظة في متصفح المستخدم فقط في هذه المرحلة. لا تضع أي مفاتيح أو أسرار داخل المستودع؛ ملفات `.env` و`.env.local` مستثناة من Git.

## الاتجاه القادم

1. استيراد وتصدير نسخة احتياطية مشفرة.
2. مرفقات محلية وروابط أدلة منظمة.
3. AI محلي اختياري عبر WebLLM للأجهزة القادرة.
4. مزامنة اختيارية عبر Backend مجاني عند الحاجة للاستخدام عبر أكثر من جهاز.
